# Health Tracker

A Flutter app for tracking health metrics including steps, sleep, calories, and cycle data.

## Features

- Track daily steps, sleep patterns, calories burned, and menstrual cycle
- Interactive charts with weekly/monthly views
- Clean, iOS-style interface
